# **Robotничкофф** - это не просто парсер, а полноценный кроссплатформенный (Windows & macOS) ассистент для тех, кто хочет найти идеальную работу, не тратя жизнь на бесконечный скроллинг сайтов. 
Больше не нужно вручную обновлять вкладки. Радар работает за тебя, собирая самые свежие предложения в единый, чистый и умный поток.

## 🚀 Ключевые фишки

* **🔍 Мульти-поиск:** Одновременный сбор и парсинг вакансий с 8 ведущих площадок: **HH.RU**, **Habr Career**, **SuperJob**, **Zarplata.ru**, **GeekJob**, **ТрудВсем**, **Remote-Job** и **Rabota.ru** в едином чистом потоке.
* **🌍 Глобальная география:** Поиск работы по всему миру. Полное покрытие России (от Калининграда до Владивостока), стран СНГ, Европы и США, а также главных хабов релокации в **Азии** (Бали, Пхукет) и **MENA** (Дубай, Стамбул, Израиль).
* **💰 Мгновенные фильтры ЗП и формата:** Тонкая настройка под ваши запросы. Ищите только **Удаленку**, **Гибрид** или **Офис**, и отсекайте вакансии «по договоренности» одним кликом. Фильтры применяются на лету, без перезапуска поиска.
* **🇬🇧 Мультиязычный интерфейс:** Полная поддержка английского и русского языков интерфейса. При этом данные внутри карточек (названия вакансий и компаний) честно сохраняют оригинальный язык для максимальной точности.
* **🥷 Фоновый радар (Smart Polling):** Сверните приложение в системный трей, и ассистент будет сам опрашивать площадки каждые 3 минуты. При обнаружении свежих вакансий вы получите **нативный Push и звуковой сигнал**.
* **🧠 Семантика и Автоисправление:** Модульная система понимает IT-контекст (знает, что Frontend — это React), а встроенный алгоритм на лету исправляет забытую раскладку клавиатуры (например, `vtytl;th` превратится в `менеджер`).
* **🛡 Умный парсинг и Анти-дубликаты:** Радар автоматически очищает ссылки от UTM-меток, не допускает повторных уведомлений об одной вакансии и безошибочно вытягивает реальные цифры зарплаты из хаоса HTML-тегов.
* **🎲 Алгоритм «Хаос»:** Уникальное перемешивание результатов обеспечивает честную выдачу со всех сайтов сразу, избавляя от монополии одного ресурса.
* **💾 Локальная память:** Ваше «Избранное» и «История просмотров» хранятся строго локально на вашем устройстве. Ранее открытые карточки визуально приглушаются, чтобы не отвлекать внимание.
* **🎨 Безупречный UI/UX:** Темная и светлая темы, плавные анимации, запуск поиска по нажатию **Enter** и "прозрачные" настройки, позволяющие видеть изменения ленты в реальном времени.

## Технологический стек
* **Frontend:** React + TypeScript + Tailwind CSS - молниеносный и плавный интерфейс.
* **Backend:** Rust (Tauri) - максимальная производительность, безопасность и ничтожное потребление оперативной памяти.

## Установка и запуск
### Windows
1. Скачайте файл `.msi` или `.exe` в разделе [Releases](https://github.com/theonestory/robotnichkoff/releases/latest).
2. Установите и запустите.

### macOS (Apple Silicon / Intel)
1. Скачайте `.dmg` из последнего релиза.
2. Перенесите **Robotnichkoff** в папку `Applications`.
3. При первом запуске выполните команду в Терминале:
   ```bash
   xattr -cr /Applications/Robotnichkoff.app
