# 🤖 Robotничкофф

**Robotничкофф** — это быстрый и легкий десктопный агрегатор вакансий, созданный для тех, кто хочет видеть полную картину рынка труда без бесконечного переключения вкладок. Приложение собирает данные с ключевых площадок и объединяет их в единый, удобно читаемый поток.

## ✨ Основные фишки
* 🔍 **Мульти-поиск:** Одновременный парсинг вакансий с **HH.RU, Habr Career, SuperJob и Zarplata.ru**.
* 🎲 **Алгоритм «Хаос»:** Уникальное перемешивание результатов. Вы видите честную выборку со всех сайтов сразу, а не монопольный список одного ресурса.
* 🥷 **Фоновый радар (Smart Polling):** Приложение можно свернуть в системный трей. Ассистент будет сам опрашивать площадки каждые 3 минуты и присылать **нативные Push-уведомления** о новых вакансиях, пока вы занимаетесь своими делами.
* 🧠 **Умный семантический фильтр:** Жесткая защита от спама. Алгоритм понимает разницу между `CPO` и `Product Owner`, знает словарь IT-синонимов (Frontend = React) и отсекает нерелевантный мусор.
* 🛡 **Анти-дубликаты:** Smart Parsing автоматически срезает UTM-метки и динамические хвосты из ссылок. Радар никогда не пришлет вам пуш на одну и ту же вакансию дважды.
* 💾 **Локальная память и История:** Ваше «Избранное» хранится прямо на устройстве (без регистраций). А «История просмотров» запоминает клики и удобно приглушает уже открытые карточки в ленте поиска.
* 🎨 **Продуманный UI/UX:** Нативная навигация, умная «плавающая» кнопка новых вакансий (не ломает скролл) и переключение **Светлой / Тёмной темы** на лету.
* 🚀 **Автообновления:** Приложение само проверяет новые релизы на GitHub и позволяет обновиться в один клик.

## 🛠 Технологический стек
Проект построен на современных и максимально производительных технологиях:

* **[Rust](https://www.rust-lang.org/):** Мощный и безопасный язык для бэкенда. Обеспечивает молниеносную скорость парсинга.
* **[Tauri 2.0](https://tauri.app/):** Фреймворк нового поколения для десктопных приложений. Минимум потребления памяти, максимум безопасности.
* **[React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/):** Современный фронтенд с жесткой типизацией.
* **[Tailwind CSS](https://tailwindcss.com/):** Для создания чистого и адаптивного дизайна.
* **[GitHub Actions](https://github.com/features/actions):** CI/CD конвейер для автоматической сборки версий под Windows и macOS в облаке.
* **[Gemini AI](https://gemini.google.com/):** Искусственный интеллект от Google выступил в роли архитектора, эксперта по Rust и DevOps-инженера, помогая реализовать кроссплатформенную сборку.

## 🍏 Инструкция для пользователей macOS

Так как приложение не имеет платной подписи Apple, система блокирует его запуск. Чтобы снять ограничение «карантина», выполните одну команду в Терминале:

1. Перетащите **Robotnichkoff.app** в папку **Программы** (Applications).
2. Откройте **Терминал** (через Spotlight: `Cmd + Пробел` -> `Terminal`).
3. Скопируйте и вставьте следующую команду, затем нажмите **Enter**:
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/Robotnichkoff.app
