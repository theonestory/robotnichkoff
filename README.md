# 🤖 Robotничкофф (Robotnichkoff)

**Robotничкофф** — это быстрый и легкий десктопный агрегатор вакансий, созданный для тех, кто хочет видеть полную картину рынка труда без бесконечного переключения вкладок. Приложение собирает данные с ключевых площадок и объединяет их в единый, удобно читаемый поток.

## ✨ Основные фишки
* **Мульти-поиск:** Одновременный парсинг вакансий с **HH.RU**, **Habr Career**, **SuperJob** и **Zarplata.ru**.
* **Алгоритм «Хаос»:** Уникальное перемешивание результатов. Вы видите честную выборку со всех сайтов сразу, а не монопольный список одного ресурса.
* **Smart Parsing:** Автоматическая очистка данных. Мы вырезаем лишний текст из зарплат и корректно отображаем названия компаний.
* **Тёмная тема:** Премиальный дизайн для комфортной работы в любое время суток.
* **Локальное Избранное:** Сохраняйте интересные вакансии прямо в приложении без регистрации.

## 🛠 Технологический стек (The Tech Stack)
Проект построен на современных и максимально производительных технологиях:

* **[Rust](https://www.rust-lang.org/):** Мощный и безопасный язык для бэкенда. Обеспечивает молниеносную скорость парсинга.
* **[Tauri 2.0](https://tauri.app/):** Фреймворк нового поколения для создания десктопных приложений. Минимум потребления памяти, максимум безопасности.
* **[React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/):** Современный фронтенд с жесткой типизацией для стабильной работы интерфейса.
* **[Tailwind CSS](https://tailwindcss.com/):** Для создания чистого и адаптивного дизайна.
* **[GitHub Actions](https://github.com/features/actions):** Настроенный CI/CD конвейер для автоматической сборки версий под Windows и macOS в облаке.
* **[Gemini AI](https://gemini.google.com/):** Искусственный интеллект от Google выступил в роли архитектора, эксперта по Rust и DevOps-инженера, помогая реализовать кроссплатформенную сборку и логику парсинга.

## 📥 Скачать (Download)
Актуальные версии для вашей операционной системы всегда доступны в разделе **[Releases](https://github.com/theonestory/robotnichkoff/releases/latest)**:
