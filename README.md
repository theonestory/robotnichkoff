# 🚀 **Robotничкофф** - это не просто парсер, а полноценный кроссплатформенный (Windows & macOS) ассистент для тех, кто хочет найти идеальную работу, не тратя жизнь на бесконечный скроллинг сайтов. 
Больше не нужно вручную обновлять вкладки. Радар работает за тебя, собирая самые свежие предложения в единый, чистый и умный поток.

## ✨ Ключевые фишки

* **🔍 Мульти-поиск:** Одновременный парсинг вакансий с крупнейших площадок: **HH.RU, Habr Career, SuperJob и Zarplata.ru** в одном окне.
* **🌍 Умная география:** Поддержка 50+ крупнейших IT-хабов СНГ (Россия, Казахстан, Беларусь, Узбекистан, Таджикистан). При поиске по крупным городам радар автоматически захватывает прилегающие области (например, Москва + МО), чтобы не упустить скрытые вакансии.
* **🏢 Фильтры формата работы:** Тонкая настройка под ваш стиль жизни — ищите только **Удаленку**, **Гибрид** или работу в **Офисе**.
* **🎲 Алгоритм «Хаос»:** Уникальное перемешивание результатов обеспечивает честную выдачу со всех сайтов сразу, избавляя от монополии одного ресурса.
* **🥷 Фоновый радар (Smart Polling):** Сверните приложение в трей, и ассистент будет сам опрашивать площадки каждые 3 минуты, присылая **нативные Push-уведомления** о новых вакансиях.
* **🧠 Семантическая фильтрация:** Модульная система фильтров понимает IT-контекст (знает, что Frontend — это React) и отсекает нерелевантный мусор.
* **🛡 Анти-дубликаты:** Smart Parsing автоматически очищает ссылки от UTM-меток и динамических параметров, исключая повторные уведомления об одной и той же вакансии.
* **💾 Локальная память:** Ваше «Избранное» и «История просмотров» хранятся только на вашем устройстве. Приложение визуально приглушает карточки, которые вы уже открывали.
* **🎨 Продуманный UI/UX:** Режим «Абсолютного фокуса» для настройки фильтров, темная и светлая темы, а также мгновенный запуск поиска по нажатию **Enter**.

## 💻 Технологический стек
* **Frontend:** React + TypeScript + Tailwind CSS - молниеносный и плавный интерфейс.
* **Backend:** Rust (Tauri) - максимальная производительность, безопасность и ничтожное потребление оперативной памяти.

## 📥 Установка и запуск
### 🪟 Windows
1. Скачайте файл `.msi` или `.exe` в разделе [Releases](https://github.com/theonestory/robotnichkoff/releases/latest).
2. Установите и запустите.

### 🍎 macOS (Apple Silicon / Intel)
1. Скачайте `.dmg` из последнего релиза.
2. Перенесите **Robotnichkoff** в папку `Applications`.
3. При первом запуске выполните команду в Терминале:
   ```bash
   xattr -cr /Applications/Robotnichkoff.app
