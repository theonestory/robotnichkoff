# 🚀 **Robotничкофф** — это не просто парсер, а полноценный кроссплатформенный (Windows & macOS) ассистент для тех, кто хочет найти идеальную работу, не тратя жизнь на бесконечный скроллинг сайтов. 
Больше не нужно вручную обновлять вкладки. Радар работает за тебя, собирая самые свежие предложения в единый, чистый и умный поток.

## ✨ Ключевые фишки
* **🔍 Мульти-поиск:** Одновременный парсинг вакансий с крупнейших площадок: **HH.RU, Habr Career, SuperJob и Zarplata.ru**.
* **🎲 Алгоритм «Хаос»:** Уникальное перемешивание результатов. Вы видите честную выборку со всех сайтов сразу, а не монопольный список одного ресурса.
* **🥷 Фоновый радар (Smart Polling):** Сверните приложение в трей, и оно будет опрашивать площадки каждые 3 минуты, присылая **нативные Push-уведомления** о новых вакансиях, пока вы заняты делами.
* **🧠 Умный семантический фильтр:** Жесткая защита от нерелевантных объявлений. Алгоритм понимает разницу между CPO и Product Owner, знает словарь IT-синонимов (Frontend = React) и отсекает мусор.
* **🌍 Глобальный охват:** Поддержка 50+ крупнейших IT-хабов СНГ (Россия, Казахстан, Беларусь, Узбекистан, Таджикистан) с автоматическим захватом областей.
* **🛡 Анти-дубликаты:** Smart Parsing автоматически срезает UTM-метки и динамические хвосты ссылок. Радар никогда не пришлет уведомление об одной и той же вакансии дважды.
* **💾 Локальная память и История:** Ваше «Избранное» хранится только на устройстве (без регистраций). «История просмотров» запоминает клики и удобно приглушает уже открытые карточки в ленте.
* **🎨 Продуманный UI/UX:** Нативный интерфейс с режимом «Абсолютного фокуса», плавающей кнопкой новых вакансий (не ломает скролл) и переключением **Светлой / Тёмной темы** на лету.
* **🚀 Автообновления:** Приложение само проверяет новые релизы на GitHub и позволяет обновиться в один клик.

## 💻 Технологический стек
* **Frontend:** React + TypeScript + Tailwind CSS — молниеносный и плавный интерфейс.
* **Backend:** Rust (Tauri) — максимальная производительность, безопасность и ничтожное потребление оперативной памяти.

## 📥 Установка и запуск
### 🪟 Windows
1. Скачайте файл `.msi` или `.exe` в разделе [Releases](https://github.com/theonestory/robotnichkoff/releases/latest).
2. Установите и запустите.

### 🍎 macOS (Apple Silicon / Intel)
1. Скачайте `.dmg` из последнего релиза.
2. Перенесите **Robotnichkoff** в папку `Applications`.
3. При первом запуске выполните команду в Терминале:
   ```bash
   xattr -cr /Applications/Robotnichkoff.app
